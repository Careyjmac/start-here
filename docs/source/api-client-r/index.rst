The R client
------------

The `api-client-r <https://github.com/googlegenomics/api-client-r>`_ project
provides an R package with methods to search for Reads and Variants stored in the Google Genomics API.  Additionally it provides converters to BioConductor datatypes such as `GAlignments <http://www.bioconductor.org/packages/release/bioc/html/GenomicAlignments.html>`_, `GRanges <http://www.bioconductor.org/packages/release/bioc/html/GenomicRanges.html>`_, and `VRanges <http://www.bioconductor.org/packages/release/bioc/html/VariantAnnotation.html>`_.
