The R client
------------

The `api-client-r <https://github.com/googlegenomics/api-client-r>`_ project
provides an R package with methods to search for Reads and Variants stored in the Google Genomics API.  Additionally it provides converters to BioConductor datatypes such as GAlignments, GRanges, and VRanges.
